# Typography

Our typography system is designed to create clear visual hierarchy and ensure readability across all platforms.

## Font Family

We use system fonts for optimal performance and native feel:

```css
font-family: system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue',
  Arial, sans-serif;
```

## Usage Guidelines

- Use appropriate font sizes for different screen sizes
- Maintain consistent line heights for readability
- Use font weights to create visual hierarchy
- Ensure sufficient contrast between text and background
