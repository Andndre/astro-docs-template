## Overview

The Button component is a versatile UI element used for initiating actions or submitting data in your application. It is one of the most common interactive elements and is designed to be highly customizable to meet different use cases.

## Usage

The Button component should be used to perform primary or secondary actions within your UI. It is important to choose the correct button style depending on the importance of the action.

## Props

| Prop     | Type                                   | Default   | Description                            |
| -------- | -------------------------------------- | --------- | -------------------------------------- |
| variant  | 'primary' \| 'secondary' \| 'tertiary' | 'primary' | The visual style variant of the button |
| size     | 'small' \| 'default' \| 'large'        | 'default' | The size of the button                 |
| disabled | boolean                                | false     | Whether the button is disabled         |
| type     | 'button' \| 'submit' \| 'reset'        | 'button'  | The HTML button type                   |

## Accessibility

- Buttons are focusable and can be triggered using both mouse and keyboard
- When disabled, buttons cannot be focused and clearly show their disabled state
- The component supports ARIA attributes through props
