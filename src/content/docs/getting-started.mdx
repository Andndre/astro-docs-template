# Getting Started

Welcome to the Acme Inc. Design System. This guide will help you get up and running with our components and design tokens.

## Installation

1. Install the package using npm:

```bash
npm install @acme/design-system
```

2. Add the required dependencies:

```bash
npm install tailwindcss @sveltejs/kit
```

3. Import our Tailwind configuration:

```js
// tailwind.config.js
import acmePreset from '@acme/design-system/preset'

/** @type {import('tailwindcss').Config} */
export default {
  presets: [acmePreset],
  // Your custom configuration...
}
```

## Usage

Import and use our components in your Svelte files:

```svelte
<script>
  import { Button } from '@acme/design-system'
</script>

<Button variant="primary">Click me</Button>
```

## Development

To contribute to the design system:

1. Clone the repository
2. Install dependencies
3. Start the development server
4. Make your changes
5. Submit a pull request
