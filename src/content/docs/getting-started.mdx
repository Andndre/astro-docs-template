Get up and running with our design system in your Svelte project. This guide will walk you through installation, basic usage, and best practices.

## Quick install

Add our design system to your project:

```bash
# Using npm
npm install @acme/design-system

# Using pnpm
pnpm add @acme/design-system
```

## Basic setup

1. Configure Tailwind CSS in your project:

```js
// tailwind.config.js
import acmePreset from '@acme/design-system/preset'

/** @type {import('tailwindcss').Config} */
export default {
  content: ['./src/**/*.{html,js,svelte,ts}'],
  presets: [acmePreset],
}
```

2. Import our base styles in your root layout:

```js
// src/routes/+layout.js
import '@acme/design-system/styles.css'
```

## Using components

Import and use components in your files:

```js
// Example component usage
import { Button } from '@acme/design-system'

// Primary button
<Button variant="primary">
  Get Started
</Button>

// Secondary button
<Button variant="secondary">
  Learn More
</Button>
```

## Dark mode support

Our components support dark mode out of the box. Enable it in your app:

```js
// Toggle dark mode
document.documentElement.classList.toggle('dark')

// Check current theme
const isDark = document.documentElement.classList.contains('dark')
```

## Troubleshooting

Common issues and solutions:

- **Styles not working?** Ensure Tailwind is configured correctly and processing your files
- **TypeScript errors?** Check you have `@sveltejs/kit` and `typescript` installed
- **Components not found?** Verify the import path and package installation
